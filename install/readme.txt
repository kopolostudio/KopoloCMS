Порядок работы установщика:
*Проверка сервера на совместимоть с системой
*Копирование содержимого confi_sample.php в config.php, с подстановкой параметров соединения с базой данных
*Загрузка дампа из install/sql/default.sql, с подменой %db-prefix% на префикс БД

Для активации установщика необходимо, чтобы config.php отсутствовал или был меньше 500 байт размером, однако это не повод оставлять директорию install/. Если удалять директорию инсталл не хочется, можно раскомментировать Deny в .htaccess
